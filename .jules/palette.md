## 2025-01-24 - Form inputs lacked helpful hints
**Learning:** Adding subtle placeholder text to empty filter inputs significantly improves context, especially for fields requiring UUIDs or specific formats. Users often pause on these fields if they aren't sure what's expected.
**Action:** Always include a `placeholder` attribute when creating or modifying input fields, even for internal or technical views, to guide the user on the expected format (e.g. `e.g. 123e4567...`).
