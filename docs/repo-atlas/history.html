intentionally deferred

Boundaries that prevent accidental scope creep.

No Kubernetes yet

Single-host Compose and systemd are the supported paths; add orchestration only when a deployment needs it.

No RBAC platform yet

Forge is trusted, single-tenant v0.1. Workspace membership is a foundation, not complete authorization.

No native OTel endpoint yet

Health endpoints, structured logs, and scoped durable webhook export cover the stated baseline.
